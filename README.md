# 说明

> 简单的一个vue2 + vuex + axios + element-ui + vue-router +mockjs +less。

>  开发环境  nodejs 6.10.0

>  这个项目主要用于vue2 + vuex + axios + vue-router 拦截  


## 项目运行（nodejs 6.0+）
``` bash
# 克隆到本地
git clone https://github.com/fengqz/game.git
# 安装依赖
npm install

# 安装依赖
npm i mockjs -g
# 安装依赖
npm i json-server -g

# 开启本地服务器localhost:8088
npm run mockdev

# 发布环境
npm  run build
```
